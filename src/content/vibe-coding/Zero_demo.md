零之度是一个面向 AI 产品演示的前端体验中心。

它不接后端、不连数据库、不做登录 —— 用纯前端的 mock 数据展示不同业务场景下 AI 产品的交互形态。

## 为什么做

教育信息化产品的采购决策链长，涉及教育局、学校、老师、家长各方。光靠 PPT 和原型图很难让各方真正理解产品。做可交互的 Demo，是让 stakeholders 自己上手体验的最短路径。

## 覆盖场景

- **教师工作台** — 今日待办、课表、班级档案下钻、家校协同任务闭环
- **家长端 APP** — iPhone 容器内的家校协同，消息反馈 + 老师端状态回流
- **集团治理后台** — 政策闭环、智慧课堂预警、校园声量监测、风险事件合并
- **业务数据支撑** — 展示业务方需提供的数据字段、标准和采集方式
- **政策任务中心** — 政策文件解析、多校分析、任务分发和进度追踪

所有页面都是用 React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui 搭建的。

## 当前状态

体验展示阶段，多个领域的交互流程已经跑通，可以作为后续真实业务系统的交互规范参考。
