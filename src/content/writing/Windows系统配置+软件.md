老电脑服役 7 年退役了。新机到手第一件事不是装游戏，是把开发环境和日常软件恢复到正常状态。

记录一下，方便以后自己用，也给别人参考。

## 1. 系统配置

- **系统名称**：设置 → 主页 → 更改电脑名称
- **个性化**：主题颜色 + 壁纸 + macOS 风格鼠标指针
- **任务栏**：关闭/隐藏不需要的任务项，托盘图标打开需要的，任务栏行为选"从不合并"
- **更新**：Windows Update 走一遍，开到最新
- **文件夹选项**：打开"此电脑"、显示隐藏文件、显示文件后缀名

## 2. 软件清单

**系统工具：**
Clash / UU 加速器、Chrome + Edge、MacType（改善字体渲染）、Logitech Options、GeekUninstaller、思源系列字体

**生产力工具：**
Office 365 家庭版 + Visio、Axure RP9、Xmind、Typora、VS Code、Git + Node.js + Hexo

**日常软件：**
微信 / QQ / 钉钉 / 腾讯会议、Spotify / QQ 音乐、115 网盘 / 百度网盘、Eagle、NeatReader、PotPlayer、IDM

**游戏：**
Battle.net、Steam、完美世界对战平台、5E 对战平台

这份清单现在已经过时了——我主力切到了 Mac。但 Windows 那 7 年的服役记录说明了一件事：**好用的软件不需要频繁换。找到一个顺手的，用下去就好。**
