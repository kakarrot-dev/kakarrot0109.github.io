No. 2: 魔兽世界与产品设计

从艾泽拉斯的插件DIY出发，聊一个产品经理的自我修养。玩游戏不只是玩游戏。

2014/7/1

产品设计魔兽世界成长个人叙事

2008 年，高二。

每周逃课去网吧，跟同学组队做任务。死矿、影牙、血色、祖尔、斯坦索姆。那时候不知道什么是插件，只知道月光宝盒和魔兽精灵能把背包一起打开、能帮我更快完成任务。

那时候的我以为，这就是这个游戏的全部。

直到遇见庸医。

1️⃣ 第一次知道“可以不一样”

高三毕业，魔兽正经历网易取代九城代理的关服期。时间充裕，开始逛贴吧。在牧师吧看到了庸医的插件包。

第一次知道玩家和目标的头像可以放到屏幕中间，还不影响视野。

第一次知道动作条、小地图、聊天框、伤害显示可以美化的那么酷。

第一次知道技能提醒、冷却监控、伤害治疗统计、仇恨统计——这些概念。

震撼。下载他的插件包和教程，开始研究。跟庸医交流时，他还把我的 ID 写进了他的教程文档里。高中毕业的小屁孩，看到自己的名字出现在别人的作品里——那种心情很难描述。

之后的生活就是天天逛 NGA、泡插件社区、问度娘。弄明白了 Interface / Addons / WTF 的关系，知道了大众用的单体插件有哪些，学会了基本配置。

一发不可收拾。

2️⃣ 插件告诉我什么是“极致”

高三暑假是我魔兽生涯的第一个巅峰。跟工会打通 HS BT SW，自己开国家队战场小队，YY 1214 联盟奥山指挥。还认识了很多朋友，帮他们解决一个又一个插件问题。

到大学开 WLK 时，之前的朋友们说好的回归也没人了。我练到 80 级就 AFK 了。高三暑假看国外插件吊炸天，想着开 WLK 后自己做一套，也没如愿。

错过一个巅峰版本的魔兽是不完整的。 但我几次回归，都不是为了装备，而是为了倒腾插件。

CTM 时代玩着小号法师战场 PVP，没事跟朋友折腾插件。NGA 有个精品贴收集了 CTM 时代的整合包，每个都用过。

MOP 时代开始工作了，但没停止倒腾插件。看视频、看直播，学习别人的界面摆放和功能设置。后来接触了 Twitch，发现 Reckful、venruki、Watchmeblink 这些世界大神的 PVP 界面并不花里胡哨——用的是自带界面。我又纠结了。插件几乎全删，只留了 JJC 框架。然后学怎么调血量按 K 显示、怎么修改 lua。直到设置出自己满意的界面才罢休。

3️⃣ 从玩插件到做产品

现在回头看，那段倒腾插件的经历，对我做产品经理的帮助超出想象。

界面摆放、对称、美观、视野控制——这些在 2009 到 2014 年间积累的经验，后来全用在了产品设计上。设计软件和网站的 UI、功能、需求时，那些“看起来舒服”的判断，很多来自魔兽插件时代的直觉。

写这些不是为了炫耀玩了多久。是想说一件事：当你真正喜欢一个东西的时候，不用别人逼你，你就会自发地去学习各种知识。

玩游戏不是只玩游戏本身。那你就是被游戏玩了。玩游戏交朋友、学东西，才是我玩游戏的目的。

产品经理的审美不是上课学来的。是在某些你完全没意识到的时刻，一点点积累的。

那些你以为在“浪费时间”的事情，也许正在为你的未来打基础。只是你不知道而已。

在你看不见的地方训练自己，才是最好的训练。
