No. 7: 产品营销那些事

产品经理不只管功能。市场和营销思维，是做产品时必须带上的另一副眼镜。

2016/7/9

产品管理市场营销产品经理竞争分析

产品经理容易犯一个毛病：觉得“营销是市场部的事”。

我以前也这么想。直到有一个产品上线后数据惨淡，我跟市场同事坐下来一聊，才发现：我做的功能假设，跟市场认为的用户需求根本不在一个频道上。

从那之后我意识到——产品经理必须带着营销思维做产品。 不是让你去跑渠道，是在设计阶段就想清楚谁会买、为什么买、怎么被找到。

1️⃣ 市场分析

市场分析不是大公司的专利。小型产品团队同样需要，区别只是分析深度。

核心是顾客细分——把一群有共同需求、共同特征的用户拆开来看。拆得越细，产品做起来越聚焦。

三个价值：

  • 了解顾客怎么买、为什么买
  • 更有针对性地分配资源，建立竞争优势
  • 发现被忽略的市场机会，找到蓝海入口

2️⃣ 竞争分析

竞争分析不是让你抄竞品，是让你知道：你输给过谁、赢过谁。很多人只盯着行业老大看，但真正抢你饭吃的可能是你压根没注意到的对手。

围绕几个维度展开：

用户：

  • 目标用户在哪
  • 他们有什么共性
  • 注册用户中活跃 / 流失 / 付费 / 僵尸的占比

产品：

  • 成本多少，预期何时持平
  • 产品名称能不能当品牌用
  • 有什么特性能让用户区分你

销售线索：

  • 团队结构能不能支撑目标
  • 用了哪些渠道触达用户
  • 是否帮用户理解产品价值

定价：

  • 定价是否把成本和竞争定位都考虑了
  • 有没有因为价格流失用户

3️⃣ 外部环境

行业在变。影响产品的不仅仅是竞品，还有科技变化、法规政策、经济环境。

几个值得追问的问题：

  • 未来几年哪些技术变化会冲击你的产品
  • 行业的价格政策、分销方式有没有变化
  • 有没有潜在并购或撤资风险
  • 当前的经济周期对产品在线是利好还是利空

这些问题不是让你去预测未来。是做产品的人应该养成定期扫一圈外部环境的习惯——大多数时候你扫了也没用，但偶尔扫到的那一次，能救你一条命。

2016 年写这篇文章时我还在摸索。那会儿的笔记很教科书写法，现在回头看有些生硬。但核心意思没变——产品经理的视野不能只停在原型图上。市场、用户、定价、渠道，这些离“功能”很远的东西，最终都会决定你的产品能走多远。
